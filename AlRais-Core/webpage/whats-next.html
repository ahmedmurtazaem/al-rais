Part three · Tunable vs structural

What takes minutes. What takes weeks.

A request to change a markup formula or a curation label is a configuration change measured in minutes. A request to add a new supplier type or alter the canonical data model is engineering work measured in weeks. Knowing which is which upfront prevents surprises in either direction.

Tunable
Configuration · minutes
Takes effect immediately across the website, chatbot, recommendation engine and admin panel.
Structural
Engineering · weeks
Deliberate architectural decisions. Their stability is what makes everything else reliable and extensible.
Read this as evidence of deliberate design, not limitation. The platform is built to absorb commercial change through configuration, and to absorb supplier growth through its connector pattern. What is fixed is fixed because keeping it fixed is what keeps everything else flexible.
